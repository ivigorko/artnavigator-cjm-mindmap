# Mind Map педагога АртНавигатор — Инструкция по деплою

## Что готово

`index.html` (105 KB, автономный HTML, без CDN-зависимостей). Содержит:
- Полный путь педагога по сегменту "Массовый" — 38 шагов с реальными деталями
- Touchpoint, Goal, мысли педагога, эмоции, боли, возможности для каждого шага
- Брендинг АртНавигатор + слоган с сайта
- Адаптивный layout: слева timeline, справа детали шага

## 3 способа задеплоить

### Способ 1: Netlify Drop (самый быстрый)
1. Открыть https://app.netlify.com/drop
2. Перетащить папку `MindMap_Pedagog_deploy` (или только `index.html`)
3. Получить URL вида `https://random-name-123.netlify.app`
4. Готово — занимает 30 секунд

### Способ 2: Netlify CLI (для автообновлений)
```bash
npm install -g netlify-cli
netlify login
netlify deploy --prod --dir=./MindMap_Pedagog_deploy
```

### Способ 3: GitHub + Netlify (для версионирования)
1. Создать репозиторий на GitHub
2. Подключить к Netlify через UI
3. Каждый push → автодеплой

## Структура файла

`index.html` — единый файл со всем:
- HTML разметка
- CSS стили (inline `<style>`)
- JavaScript логика (inline `<script>`)
- Данные 38 шагов (inline JSON в `<script type="application/json">`)

Никаких внешних зависимостей. Работает офлайн.

## Локальный просмотр

Можно открыть `index.html` напрямую в браузере (двойной клик в проводнике). Без деплоя на сервер работает.
