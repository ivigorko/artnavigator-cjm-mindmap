# Mind Map педагога АртНавигатор

Интерактивная Customer Journey Map педагога: дерево «сегмент — персона — горизонт — фаза» с боковой панелью деталей, поиском, зумом и фильтрацией.

**Версия данных: CJM v2.0 · обновлено 21.07.2026**

## Что изменилось в v2.0

Карта пересобрана под обновлённые CJM. Предыдущая версия (07.07.2026) описывала три сегмента и локальный сценарий как основной.

1. **Сегмент «Базовый · Ольга» удалён** — исключён из CJM по запросу клиента.
2. **Выездной сценарий стал главным**, локальный — побочным. Фазы с выездными шагами помечены оранжевым чипом «выездное», в панели деталей выводится пояснение. Основание: основная выручка формируется не оргвзносом (от 3 000 ₽), а услугами гостеприимства — отели, билеты, трансфер, питание, 50-200 тыс. ₽ на семью за поездку.
3. **Персоны актуализированы.** Массовый: Елена Викторовна, 45-52, ДК или школа искусств в городе 100-500 тыс. Продвинутый: Светлана, 38, студия contemporary dance, Москва.
4. **Реестр фрикций, opportunity map и roadmap переписаны под v2.0.** Добавлена группа «★ Выездные · приоритет v2.0» (13 задач). Roadmap: Q3 2026 — выездные фичи, Q4 2026 — локальные, Q1-Q2 2027 — стратегические ставки, Q3 2027+ — PRO и монетизация выездного.
5. **Счётчики в шапке** считаются от данных, добавлен счётчик выездных фаз.

Состав карты: 2 сегмента, 6 горизонтов, 38 фаз (4 выездные), 29 фрикций, 33 возможности, 4 квартала roadmap.

## Структура

`index.html` — автономный файл: разметка, стили и логика внутри. Единственная внешняя зависимость — шрифт Montserrat с Google Fonts (как и в предыдущей версии). Открывается двойным кликом, работает офлайн (без интернета подставится системный шрифт).

Ветка «Стратегия и решения» по умолчанию скрыта — так было и в предыдущей версии. Чтобы показать её, удалите в `index.html` строку:

```js
DATA.children = DATA.children.filter(c=>c.kind!=='strategy');
```

## Как обновлять дальше

Карта собирается из исходных CJM тремя скриптами, править HTML руками не нужно:

```bash
python3 parse_cjm.py      # CJM_Pedagog_Massovy.md + _Prodvinuty.md -> cjm_data.json
python3 gen_phases.py     # агрегация 82 шагов в 38 укрупнённых фаз (проверка глазами)
python3 build_repo_map.py # index_ORIGINAL_repo.html + данные -> index.html
```

`index_ORIGINAL_repo.html` — версия из репозитория до обновления, база для патчей и точка отката.

Скрипты ожидают исходники в `CJM_Pedagogi/` соседней папки проекта. Пути заданы константами в начале каждого файла.
